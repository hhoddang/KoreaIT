함수형 컴포넌트에는 render 함수 사용 X 훅스 클래스형 컴포넌트에서 render 함수는 필수 o state , 라이프사이클 state 컴포넌트 내부에서 바꿀 수 있는 값 vue.js 에서 처럼 props 는 상위 (부모)컴포넌트에서 하위(자식) 컴포넌트로 데이터를 전달할 때 사용하고, 컴포넌트는 props를 읽기만 할 수 있다. vue.js 에서 처럼 props의 값을 변경할때는 상위(부모)컴포넌트 에서만 할 수 있다. App.js 컴포넌트에서 props를 변경하면 MyComponent.js 에서 변경된 props를 전달받아서만 사용한다 vue.js에서 처럼 하위(자식) 컴포넌트에서는 props 를 변경 할 수 없다. 리액트에는 두가지 state이 존재 1. 클래스형 컴포넌트가 가지고 있는 state 2. 함수형 컴포넌트가 가지고 있는 state
